1สภาพแวดล้อมระบบ (Environments)

แยกสภาพแวดล้อมเพื่อความปลอดภัยและคุณภาพ — พัฒนา/ทดสอบไม่กระทบ Production

สภาพแวดล้อมวัตถุประสงค์สเปก/ที่ตั้ง (กรอก)
Developmentพัฒนา/Config โดยทีม Dev 
Test / QAทดสอบฟังก์ชัน, integration test 
Staging (UAT)ผู้ใช้ตรวจรับ, ใกล้เคียง Production 
Productionใช้งานจริง 

Staging ใช้สเปกใกล้เคียง Production เพื่อให้ผล UAT สะท้อนของจริง — โดยเฉพาะการ integrate กับ สปสช./HL7 และ Data Migration จาก HosXP

2สเปกเซิร์ฟเวอร์และครุภัณฑ์

สเปกครุภัณฑ์ตามสัญญาอยู่ใน สัญญาแนบท้าย 1 — ตารางนี้สรุปองค์ประกอบหลักสำหรับการติดตั้ง

องค์ประกอบจำนวนสเปก/หมายเหตุ
Application Server CPU/RAM/Disk ตามสัญญาแนบท้าย 1
Database Server HA/สำรองข้อมูล (HA.OS)
Web / Reverse Proxy SSL/Load balance
Storage / Backup สำหรับภาพรังสี/PACS index
อุปกรณ์เครือข่าย/ความปลอดภัย Firewall/VPN เชื่อม 9 จุดบริการ

3ขั้นตอนการติดตั้ง (Installation Checklist)

  • ตรวจรับครุภัณฑ์/เซิร์ฟเวอร์ตามสัญญาแนบท้าย 1
  • ติดตั้งระบบปฏิบัติการ + patch ความปลอดภัย
  • ติดตั้งฐานข้อมูลและ middleware ที่จำเป็น (HA.OS)
  • Deploy แอปพลิเคชัน HA.OS แต่ละ environment
  • ตั้งค่า config การเชื่อมต่อ สปสช./HL7 และ Data Migration จาก HosXP
  • ตั้งค่า Backup/Restore และทดสอบกู้คืน
  • ทดสอบ Smoke Test หลังติดตั้งทุก environment และทุกจุดบริการ
  • บันทึกค่า config และส่งมอบ Installation Guide

4เครือข่ายและความปลอดภัย

  • การเข้ารหัสการสื่อสาร (TLS/SSL) ทุกช่องทาง
  • การแบ่ง network zone และกฎ Firewall สำหรับ interface สปสช./HL7 และเชื่อมโยง 9 จุดบริการ
  • การยืนยันตัวตน/สิทธิ์ (Authentication & RBAC)
  • การจัดเก็บ Log / Audit Trail ตามมาตรฐาน HA
  • การสำรองข้อมูลและแผนกู้คืน (Backup & DR)
⚠️ ข้อควรระวัง: การเปิด port/firewall เพื่อเชื่อม สปสช./HL7 และเชื่อมโยงข้อมูลระหว่าง 9 จุดบริการ ต้องประสานฝ่าย IT ของโรงพยาบาลทั้งสองนิติบุคคลล่วงหน้า

5ลำดับการ Rollout ติดตั้ง 9 จุดบริการ

ติดตั้ง รพ.หลัก ก่อน (Go-live งวด 2 ~20/01/2027) แล้วจึง Rollout สาขา/คลินิกที่เหลือจนครบ 9 จุดบริการในงวด 3 (~27/02/2027) — ปากน้ำ (สมุทรเวชกิจ) 3 จุด + ปู่เจ้าฯ 6 จุด

ลำดับจุดบริการนิติบุคคลงวด / สถานะ (กรอก)
1รพ.เมืองสมุทรปู่เจ้าฯ (รพ.หลัก)ปู่เจ้าฯงวด 2 — Go-live รพ.หลัก  
2รพ.เมืองสมุทรปากน้ำ (สมุทรเวชกิจ) (รพ.หลัก)ปากน้ำงวด 2 — Go-live รพ.หลัก  
3คลินิกเครือข่ายปู่เจ้าฯ #1ปู่เจ้าฯงวด 3 — Rollout  
4คลินิกเครือข่ายปู่เจ้าฯ #2ปู่เจ้าฯงวด 3 — Rollout  
5คลินิกเครือข่ายปู่เจ้าฯ #3ปู่เจ้าฯงวด 3 — Rollout  
6คลินิกเครือข่ายปู่เจ้าฯ #4ปู่เจ้าฯงวด 3 — Rollout  
7คลินิกเครือข่ายปู่เจ้าฯ #5ปู่เจ้าฯงวด 3 — Rollout  
8คลินิกเครือข่ายปากน้ำ #1ปากน้ำงวด 3 — Rollout  
9คลินิกเครือข่ายปากน้ำ #2ปากน้ำงวด 3 — Rollout  
คำแนะนำ: ติดตั้งและ Go-live รพ.หลักทั้งสองนิติบุคคลก่อนเพื่อพิสูจน์ระบบจริง (งวด 2) แล้วจึงทยอย Rollout คลินิกเครือข่ายตามความพร้อมจนครบ 9 จุดบริการ และยื่นตรวจรับครบทุกแห่งในงวด 3
