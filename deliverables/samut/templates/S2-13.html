2หน่วยบริการ / ห้องตรวจ / Ward (9 จุดบริการ) ผังหน่วยบริการ รพ.หลัก + สาขางานเวชระเบียน + Implement
3รายการยา / คลังยาและวัสดุ บัญชียา รพ. / งานเภสัชกรรม (HosXP)งานเภสัชกรรม + Implement
4รหัสหัตถการ / ICD-10 / ICD-9-CM รหัสมาตรฐาน / DTMU Phase 2 (ระบบฐาน)แพทย์/เวชสถิติ + Implement
5ค่าบริการ / อัตราค่ารักษา ประกาศอัตราค่าบริการ รพ.งานเรียกเก็บ + Implement
6แพทย์ / พยาบาล / บุคลากร ทะเบียนบุคลากร รพ. / งานบุคคลงานบุคคล + Implement
7Master Data Smart IPD (เตียง/Ward/eMAR) ผัง Ward / งานการพยาบาลงานการพยาบาล + Implement
8ผู้ใช้ / สิทธิ์การเข้าถึง (Role) โครงสร้างหน่วยงาน / ITทีม IT + Implement
