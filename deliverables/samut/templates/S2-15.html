1บทนำและวิธีใช้คู่มือ

1.1 ภาพรวมระบบ — HA.OS เป็นระบบสารสนเทศโรงพยาบาล (HIS) เต็มรูปแบบ รองรับ flow ผู้ป่วยครบวงจร: ลงทะเบียน/คัดกรอง → ซักประวัติ → ตรวจ/วินิจฉัย (Doctor Module) → สั่งยา/แลป/X-ray → การพยาบาล/Smart IPD (Nurse Module, eMAR) → เรียกเก็บ/การเงิน → นัดหมาย โดยโอนย้ายข้อมูลจากระบบเดิม HosXP

1.2 สัญลักษณ์ที่ใช้ในคู่มือ

  • 🖱️ การคลิก/เลือกเมนู · ⌨️ การกรอกข้อมูล · 💾 การบันทึก
  • บังคับ ขั้นตอนที่ต้องทำ · ทางเลือก ขั้นตอนเสริม
  • [ภาพประกอบ: ____] คือตำแหน่งแทรกภาพหน้าจอจริง

2สารบัญคู่มือรายโมดูล/รายบทบาท (Module & Role-based Index)

โมดูล / บทบาท (Role)หัวข้อหลักในคู่มือ
แพทย์ (Doctor Module)เปิดเวชระเบียนผู้ป่วย · ซักประวัติ · วินิจฉัย (ICD) · สั่งยา/แลป/X-ray · บันทึกแผนการรักษา · Admit/Discharge · ออกใบนัด
พยาบาล (Nurse Module / Smart IPD)คัดกรอง/ซักประวัติ · บันทึกการพยาบาล · eMAR (บริหารยา) · จัดการเตียง/Ward · ส่งต่อเวร
เวชระเบียนลงทะเบียนผู้ป่วยใหม่ · ค้นหา/แก้ไขข้อมูลผู้ป่วย · ออกเลข HN · จัดคิว (Queue & Patient Self Visit) · ตรวจสอบสิทธิ์การรักษา
เภสัชกร (Pharmacist Module)รับใบสั่งยา · ตรวจสอบ/จ่ายยา · eMAR · ตัดสต็อกคลังยาและวัสดุ · บันทึกการแพ้ยา · พิมพ์ฉลากยา
การเงิน / งานเรียกเก็บเรียกดูรายการค่ารักษา · เก็บเงิน/ออกใบเสร็จ · บันทึกสิทธิ์เบิกจ่าย · งานเรียกเก็บ/ลูกหนี้ · สรุปรายรับประจำวัน
แอดมิน (เพิ่มเติม) 

3ตัวอย่างเนื้อหาแบบทีละขั้นตอน (Step-by-step)

3.1 การเปิดเวชระเบียนผู้ป่วย (เวชระเบียน / แพทย์)

  1. คลิกเมนู “เวชระเบียน” > “ค้นหาผู้ป่วย”
  2. กรอก HN หรือ เลขบัตรประชาชน หรือ ชื่อ–สกุล แล้วกด “ค้นหา”
  3. คลิกชื่อผู้ป่วยจากผลการค้นหาเพื่อเปิดเวชระเบียน
  4. ตรวจสอบข้อมูลพื้นฐาน ประวัติแพ้ยา และประวัติการรักษาก่อนหน้า (ข้อมูลย้ายจาก HosXP)

[ภาพประกอบ: __________________________ ]

3.2 การบันทึกคำสั่งแพทย์และการพยาบาล Smart IPD (แพทย์ / พยาบาล)

  1. จากเวชระเบียนผู้ป่วยที่ Admit อยู่ คลิกแท็บ “Smart IPD”
  2. แพทย์บันทึก คำสั่งการรักษา (Doctor Order) — สั่งยา/แลป/X-ray
  3. พยาบาลรับคำสั่งใน eMAR เพื่อบริหารยาตามเวลา
  4. บันทึกสัญญาณชีพและการพยาบาลในแบบบันทึก (Nursing Note)
  5. คลิก 💾 “บันทึก” — ระบบบันทึกเข้าประวัติการรักษาทันที
  6. ทางเลือก ส่งต่อเวร (Handover) เมื่อเปลี่ยนกะ

[ภาพประกอบ: __________________________ ]

3.3 การจัดคิวและนัดหมาย (เวชระเบียน / Queue)

  1. จากเวชระเบียนผู้ป่วย คลิกเมนู “นัดหมาย” > “สร้างนัดใหม่”
  2. เลือก คลินิก/แพทย์ และ วันที่–เวลา ที่ว่าง
  3. เลือกประเภทนัด (ติดตามผล / นัดทำหัตถการต่อ)
  4. คลิก 💾 “บันทึกนัด” — ระบบส่งการแจ้งเตือนให้ผู้ป่วยตามช่องทางที่ตั้งไว้
